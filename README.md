# Voice Execution OS MVP

말이나 메모를 날짜, 프로젝트, 실행 상태로 정리하는 개인 실행 기록 앱입니다.

## 실행 방법

1. 이 폴더에서 `run_app.cmd`를 실행합니다.
2. 열린 검은 창은 앱을 쓰는 동안 닫지 않습니다.
3. PC에서는 아래 주소로 접속합니다.

```text
http://127.0.0.1:4173
```

## 주요 기능

- 음성 녹음
- 직접 텍스트 메모 저장
- 날짜별 기록 확인
- 프로젝트별 기록 분류
- 실행 대기 상태 관리
- AI 지시문 생성
- JSON 백업 다운로드 및 가져오기
- Netlify 배포 시 OpenAI 음성변환 API 연동

## 음성 텍스트 변환

Netlify 배포 환경에서 `OPENAI_API_KEY` 환경변수를 설정하면 새로 녹음한 음성이 자동으로 텍스트로 변환됩니다.

사용 모델 기본값:

```text
gpt-4o-mini-transcribe
```

다른 모델을 쓰려면 Netlify 환경변수에 아래 값을 추가합니다.

```text
OPENAI_TRANSCRIBE_MODEL=원하는 모델명
```

## 현재 제한

- 데이터는 현재 브라우저 저장소에 저장됩니다.
- 기기를 바꾸면 JSON 백업으로 옮겨야 합니다.
- 기존 음성 기록을 나중에 다시 텍스트 변환하는 기능은 아직 없습니다.
- 구글 캘린더 자동 연동은 아직 없습니다.
